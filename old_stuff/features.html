In the following we list some features of Parsec as it is right now. Keep in mind, though, that we have still quite a long way to go and do not list features that haven't been completed yet or are only planned.

  • Dazzlingly fast network game action in a 360 degree environment
  • Native Win32 (95/98/NT), MacOS, MacOS X, and Linux versions
  • Hardware rendering on 3Dfx Voodoo Graphics and OpenGL accelerators
  • Internet game play using a client/server architecture
  • Peer-to-peer mode on IPX and TCP/IP networks
  • Fully digital fast-paced soundtrack
  • Particle special effects and weapons
  • Full view panoramic outer-space background
  • Semi-transparent command-console to configure and tweak engine and game play
  • All game play variables user-configurable on-the-fly and via configuration files
