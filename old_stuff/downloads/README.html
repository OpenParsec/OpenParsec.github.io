This folder contains the file releases of the OpenParsec version 0.2.0. Here is a brief summary:

openparsec-0.2-linux-binary.tar.gz - the linux binary distribution.

openparsec-0.2.exe - the Windows installer, should work on XP, 7 and 8

openparsec-0.2.tar.bz2 - the Open Parsec source snapshot including the game assests. For full development capabilities, use the git repository located: https://github.com/OpenParsec/openparsec

openparsec-0.2-mac-binary.tar.gz - The Mac OSX binary distribution.
